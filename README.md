# zenika-extreme-startup-express-starter

This is a starter kit for the extreme startup challenge using Express.

1. Install package

```shell
npm install
```

2. Run the app

```shell
npm start
```

or

```shell
node extreme_startup_starter.js
```

3. Go to http://localhost:8080/?q=what is your name

 Should answer:
 ```
 <YOUR NAME>
 ```

4. Happy hacking!
